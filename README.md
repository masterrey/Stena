# Stena
Projeto de jogo de professores da ESPM


Vince: game design, level design, documentação e sound design
Henrique Sobral: design de interface, cenário e animações 3D
Reinaldo Ramos: programação em Unity e testes de usabilidade


Análise do game
STENA é um game que remete aos primórdios do universo dos videogames e recria a clássica
experiência do PONG. No entanto, ao invés da clássica mecânica de duas raquetes rebatendo
uma “bola quadrada” em um cenário horizontal temos uma nova visão na qual um jogador deve
proteger um núcleo de um cenário abstrato com movimentos circulares. STENA adiciona um
elemento de pinball a um clássico dos games com uma série de levels com diferentes desafios.
Objetivos de game design
STENA é um game de um player. A cada fase a velocidade da bolinha aumenta e é preciso
“sobreviver” um minuto em cada nível. O objetivo de game design é gerar apreensão e emoção
até o último segundo, quando o tempo acaba e o player passa de fase.

Palavras-chave
